How to import into Unreal Tournament

  1. In order to use your created crosshair(s) you need to import them using UnrealED.exe which is located in you Unreal Tournament System folder, run the program to get started.
  2. With UnrealED open, click the texture browser icon/button highlighted in the red box.
  3. Now click File, then Import.
  4. You can now select the files you created earlier, then click open.
    Important notes: If you are running Unreal Tournament v436 you will only be able to import .bmp images, 469 you can import both .bmp and .png, v451 has many issues with unrealEd if you can get it to run at all you can only import .bmp files.
  5. With the Import Texture window open, give your package a name, group can be left empty, and don't change the name field.
  6. Now untick "Gen MipMaps", and tick "Masked".
  7. Now you can click the OK button to import one at a time, or OK all to do them all at once.
  8. Next all you have to do is save the package by clicking the save icon or clicking File then save.
  9. You can now close UnrealED.
